Só pra relembrar

Tags que usei:

  • Code > deixar numa fonte mono-espaçada
  • Pre > copia tudo mesmo pra transcrever exatamente como está só que no site.
  • Abbr > abreviação
  • q > quote = citações simples
  • blockquote = citações com break
  • ol > criar lista ordenada
  • ul > listas não-ordenadas
  • li > pra categorizar como item de uma lista
  • dl > lista de definição
  • dt > termo
  • dd > descricao
